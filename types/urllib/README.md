# DefinitelyTyped for Urllib

## Current Urllib Version
`v0.7.6`
## Usage

```bash
npm install --save-dev @types/node-forge
```

example:
```ts
import * as forge from ".";

urllib.curl('https://example.test.com', {
  method: "GET",
  data: {
    test: 'test',
  }
})
```
